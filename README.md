# BluFocus · 品牌智能平台

BluFocus 是面向品牌出海与口碑运营的智能工作台，包含两个功能模块：

| 模块 | 目录 | 说明 |
|------|------|------|
| **PR Studio**（出海 PR 工坊） | [A/](./A) | 出海品牌 PR 稿件生成 AI Agent：品牌信息管理、本土化内容生成、多语言适配、交互式优化、文档导出 |
| **Review Monitor**（口碑监测） | [B/](./B) | 电商负面评论监测：多商品持续监测、AI 情感与维度分析、定期报告与趋势查看 |

- 使用 **A**、**B** 目录分别开发与部署两个模块；文档与包名中采用上述语义化名称。
- 各模块环境与运行方式见各自目录下的 README。
